# Rarible ExchangeV2 Contracts

## The following prerequisites are required to be installed on your system:

- NodeJS 12
- Yarn (optional)

Then run:

```sh
yarn install
chmod +x *.sh
./scripts/1-install.sh
./scripts/2-compile.sh
./scripts/3-flatten.sh
```
